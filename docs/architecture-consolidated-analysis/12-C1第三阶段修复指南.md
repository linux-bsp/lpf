## 修改清单

### 1. gpio_write_file函数（第44-68行）

**原代码：**
```c
static int32_t gpio_write_file(const char *path, const char *value) {
    int32_t fd;
    ssize_t written;
    size_t len;

    fd = open(path, O_WRONLY);
    if (fd < 0) {
        return OSAL_ERR_GENERIC;
    }

    len = strlen(value);
    written = write(fd, value, len);
    close(fd);

    if (written != (ssize_t)len) {
        return OSAL_ERR_GENERIC;
    }

    return OSAL_SUCCESS;
}
```

**修改为：**
```c
static int32_t gpio_write_file(const char *path, const char *value) {
    int32_t fd;
    int32_t written;
    uint32_t len;

    fd = OSAL_open(path, OSAL_O_WRONLY, 0);
    if (fd < 0) {
        return OSAL_ERR_GENERIC;
    }

    len = (uint32_t)strlen(value);
    written = OSAL_write(fd, value, len);
    OSAL_close(fd);

    if (written != (int32_t)len) {
        return OSAL_ERR_GENERIC;
    }

    return OSAL_SUCCESS;
}
```

### 2. gpio_read_file函数（第71-95行）

**原代码：**
```c
static int32_t gpio_read_file(const char *path, char *buffer, size_t size) {
    int32_t fd;
    ssize_t len;

    fd = open(path, O_RDONLY);
    if (fd < 0) {
        return OSAL_ERR_GENERIC;
    }

    len = read(fd, buffer, size - 1);
    close(fd);

    if (len < 0) {
        return OSAL_ERR_GENERIC;
    }

    buffer[len] = '\0';
    return OSAL_SUCCESS;
}
```

**修改为：**
```c
static int32_t gpio_read_file(const char *path, char *buffer, size_t size) {
    int32_t fd;
    int32_t len;

    fd = OSAL_open(path, OSAL_O_RDONLY, 0);
    if (fd < 0) {
        return OSAL_ERR_GENERIC;
    }

    len = OSAL_read(fd, buffer, (uint32_t)(size - 1));
    OSAL_close(fd);

    if (len < 0) {
        return OSAL_ERR_GENERIC;
    }

    buffer[len] = '\0';
    return OSAL_SUCCESS;
}
```

## 注意事项

### 1. 类型转换

**问题**：OSAL使用固定大小类型，原生使用size_t/ssize_t

**解决**：
- `size_t` → `uint32_t`
- `ssize_t` → `int32_t`
- 添加显式类型转换

### 2. 标志位映射

**OSAL标志位**：
```c
OSAL_O_RDONLY   // 只读
OSAL_O_WRONLY   // 只写
OSAL_O_RDWR     // 读写
```

**确保**：使用OSAL定义的标志位

### 3. 错误处理

**保持**：
- 原有的错误处理逻辑
- 返回值检查
- 资源清理

### 4. 文件描述符

**注意**：
- OSAL_open返回int32_t
- 负值表示错误
- 使用OSAL_close关闭
